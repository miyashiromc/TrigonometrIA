Presentación de TrigTutor IA

Aplicación del "Modelo Cualitativo Basado en Inteligencia Artificial Generativa para la Enseñanza de Trigonometría" 

Introducción: Origen y Propósito
Este proyecto, desarrollado por 

MORALES COBEÑA MIYAKO KUSHIRO, nace como la aplicación práctica de un modelo de investigación diseñado para enfrentar un desafío clave en el contexto educativo de Ecuador: el bajo rendimiento en matemáticas, particularmente en áreas como la trigonometría, evidenciado en evaluaciones internacionales.



La investigación aborda la falta de un modelo definido que guíe la integración de la 

Inteligencia Artificial (IA) generativa en software educativo. TrigTutor IA es la respuesta a este problema: una aplicación web que materializa el modelo cualitativo propuesto, sirviendo como una herramienta tecnológica para fortalecer las habilidades matemáticas de los estudiantes de educación básica superior.





Filosofía Central: Una Herramienta de Apoyo, no un Reemplazo
En línea con la tesis, la aplicación no busca sustituir la labor docente. Por el contrario, se concibe como una 

herramienta complementaria diseñada para potenciar las habilidades humanas. Su función es ofrecer un nivel de atención y práctica personalizada que a menudo es difícil de lograr en un aula tradicional, facilitando así tanto el aprendizaje del estudiante como la labor del profesor.



Funcionalidades Clave: El Modelo Cualitativo en Acción
TrigTutor IA implementa los objetivos de la tesis a través de un conjunto de funcionalidades interconectadas que utilizan una API de IA generativa para crear una experiencia de aprendizaje adaptativa y amigable.



1. Núcleo de Aprendizaje Personalizado (IA Generativa)

La funcionalidad central de la aplicación es su capacidad para consumir una API de IA generativa para crear contenido sobre la marcha. Esto permite que cada interacción del estudiante sea única y adaptada a sus necesidades.



Contenido Educativo a Demanda: El estudiante puede solicitar cualquier tema de trigonometría, y la aplicación genera instantáneamente una lección estructurada con explicaciones, fórmulas y ejemplos.


Retroalimentación Inmediata: La IA proporciona retroalimentación personalizada en ejercicios y quizzes, un recurso clave para corregir errores y afianzar el conocimiento, abordando la necesidad de atención individualizada.

2. Chat Interactivo: El Tutor Virtual

La aplicación cuenta con una interfaz de chat donde los estudiantes pueden dialogar con la IA. Pueden hacer preguntas específicas sobre un tema, y el sistema, basado en el modelo, validará la relevancia de la consulta para mantener el enfoque en la trigonometría y las matemáticas.

3. Práctica Continua y Sin Límites

Para fortalecer las habilidades, la aplicación ofrece una sección de ejercicios ilimitados. Los estudiantes pueden elegir un tema y practicar tantos problemas como necesiten, cada uno con su respectiva solución y explicación.

Las sesiones de práctica y quizzes permiten una evaluación formativa constante, ayudando al estudiante a medir su propio progreso de manera activa.

4. Entorno Personalizado y Seguimiento del Progreso

Perfiles de Usuario: Cada estudiante tiene un perfil personalizable con su nombre, avatar e información académica. Esto no solo hace la experiencia más atractiva, sino que sienta las bases para un seguimiento individualizado.

Panel de Estadísticas: La aplicación monitorea y presenta datos sobre el rendimiento del estudiante, como el tiempo de estudio por área y la precisión en los ejercicios. Esta analítica del aprendizaje es fundamental para que tanto el estudiante como el docente puedan identificar fortalezas y debilidades.

5. Recursos de Apoyo

Una sección de Recursos ofrece acceso rápido a identidades trigonométricas fundamentales y conceptos clave, funcionando como una guía de referencia constante para el estudiante.

Conclusión
TrigTutor IA no es solo una aplicación para aprender trigonometría; es la 

demostración funcional y la validación del modelo cualitativo propuesto en la tesis. Responde directamente al problema de investigación al ofrecer una herramienta tangible, basada en IA generativa, que está diseñada específicamente para mejorar las competencias matemáticas en el contexto educativo ecuatoriano, promoviendo un aprendizaje personalizado y efectivo.
